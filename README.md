# Route Quest

A graph-based delivery route optimization system developed in C++.

## Problem Statement

Determine the most fuel-efficient route for delivering packages in a weighted graph while satisfying:

- Fuel capacity constraints
- Restricted refueling stations
- Hub-to-house dependency constraints
- Minimum travel cost

## Algorithms Used

- Floyd–Warshall Algorithm
- Greedy Heuristic Search
- Path Reconstruction using Parent Matrix

## Features

- Computes all-pairs shortest paths
- Validates fuel feasibility
- Supports refueling stations
- Generates optimized delivery routes

## Tech Stack

- C++
- STL
- Graph Algorithms
